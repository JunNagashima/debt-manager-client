相殺とは？

🔄

貸し借りを相殺

お互いに貸し借りがある場合、差し引きして残高をまとめられます

💰

現金のやり取り不要

相殺を使えば、実際にお金を渡さなくても精算できます

✅

相手の承認が必要

相殺は提案後、相手が承認すると成立します

例）

あなたがAさんに ¥3,000 貸している
Aさんがあなたに ¥1,000 貸している

→ 相殺後: あなたがAさんに ¥2,000 貸し
